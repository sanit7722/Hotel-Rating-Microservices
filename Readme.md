## Getting Started
This app is mainly contains three microservices:
* ### Hotel Microservice: [Role: ADMIN]
  - **Create Hotel** 
  - **Get All Hotels** 
  - **Get Hotel Bty Id**

   
* ### Rating Microservice: [Role: NORMAL]
  - **Create Rating** 
  - **Get All Ratings**
  - **Get Ratings By User Id:**
  - **Get Ratings By Hotel Id:**
  
* ### User Microservice: [Role: NONE]
  - **Create User:** 
  - **Get All Users**
  - **Get User By Id**
